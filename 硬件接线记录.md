## 旋钮电位器

| 项目 | 记录 |
| --- | --- |
| 模块 | 旋钮电位器模块（建议使用 `10kΩ` 线性电位器） |
| 信号口 | `S17 / GPIO17` |
| 芯片通道 | ESP32-S3 `ADC2_CH6` |
| 项目用途 | 调整温度阈值，满足初中组“旋钮调整温度阈值”评分项 |
| 计划显示 | OLED 显示旋钮原始值 `XN:数值`、当前温度阈值 `YZ:数值` |
| 当前状态 | 用户已确认 `S17 / GPIO17` 信号口接线完成；待通电后验证旋转范围、Wi-Fi开启状态下采样稳定性与OLED显示 |

### 接线

```text
旋钮 OUT / S  -> N16R8 S17 / GPIO17
旋钮 VCC / V  -> N16R8 3.3V
旋钮 GND / G  -> N16R8 GND
```

- 必须使用 `3.3V` 供电，旋钮信号电压必须保持在 `0~3.3V`，不得把 `5V` 输出送入 GPIO17。
- 若模块端子标为 `G / V / S`，以实物丝印为准，不仅凭导线颜色判断。
- GPIO17 属于 ADC2；固件采用低频单次采样、滤波和有效值检查，并在 Wi-Fi 开启时复测，不能只在离线状态下验收。
- 温度阈值建议映射为 `18~35℃`；最终范围在方案确认后冻结，旋钮端点需保留少量死区，避免边缘抖动。

### 固件对应

```cpp
constexpr uint8_t PIN_TEMP_KNOB = 17;
```

### 控制侧接线

```text
继电器 S     -> N16R8 S12 / GPIO12
继电器 GND   -> N16R8 GND
继电器 VCC   -> +5V（该图片中的继电器线圈标称 5V）
```

- GPIO12 输出为 3.3V 逻辑，继电器模块必须支持 3.3V 信号触发。
- 若继电器由独立 5V 电源供电，外部电源 GND 必须与 N16R8 GND 共地。
- 图片中的黑、红、黄线可分别作为 `GND / VCC / S` 参考，但实物必须以模块丝印为准。
- 不采用参考图中疑似 `GPIO14` 的信号位置，因为本项目 `GPIO14` 已固定用于 DHT11 温湿度传感器。

### 负载侧接线

- 继电器触点只串接低压 LED 灯或模型负载。
- 当前项目禁止连接 `220V` 市电。
- 接线和改线前先断开 USB 与外部电源，检查完成后再通电。

### 固件对应

```cpp
constexpr uint8_t PIN_RELAY = 12;
```

计划固件必须将 `GPIO12` 配置为输出并在开机阶段进入安全关闭状态。接入负载前必须先做空载验证，确认该继电器模块的实际触发极性；只有实测继电器没有吸合、状态灯和遥测一致后，才能冻结安全关闭电平。
